# Proyecto Final Data Science II

## 📊 Predicción de Ventas en Retail

Este proyecto tiene como objetivo analizar el comportamiento de las ventas y desarrollar un modelo de Machine Learning para predecir la demanda.

## 🧠 Enfoque

El problema se aborda como un problema de regresión, utilizando variables temporales, promociones y comportamiento histórico de las ventas.

Se entrenaron dos modelos:
- Regresión Lineal
- Random Forest

Se seleccionó como modelo final el **Random Forest optimizado**, ya que presentó el mejor desempeño en las métricas evaluadas.

## 📁 Dataset

Para facilitar la reproducibilidad, se trabaja con un dataset filtrado correspondiente a:
- 1 tienda (`store_nbr = 1`)
- 1 categoría de productos (`GROCERY I`)

## ▶️ Cómo ejecutar

1. Abrir el notebook `.ipynb`
2. Ejecutar todas las celdas (Run All)
3. El dataset se carga automáticamente desde el repositorio
